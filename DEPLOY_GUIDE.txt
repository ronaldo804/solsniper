🚀 SOLSNIPER - دليل الرفع على GitHub و Vercel 🚀
============================================

هذا الدليل يوريك كيفاش تطلع السيت نتاعك للأنترنت باش يولي خدام 100% ويجيب لك العمولات.

المرحلة 1: إنشاء مستودع على GitHub
-----------------------------------
1. افتح حسابك في GitHub (إذا ما عندكش كريي واحد).
2. اضغط على زر "New" (أو علامة + في الفوق) باش تكريي Repository جديد.
3. سميه: solsniper
4. خليه "Public" واضغط على "Create repository".
5. راح يخرج لك رابط يشبه لهادا: https://github.com/رابط_حسابك/solsniper.git (ديرلو Copy).

المرحلة 2: رفع الملفات باستعمال الأوامر (Terminal)
-------------------------------------------------
افتح الـ Terminal في المجلد نتاعك (E:\scanar) واكتب هاد الأوامر بالترتيب:

git init
git add .
git commit -m "Initial commit: SolSniper Full Trading Engine"
git branch -M main
git remote add origin [حط هنا الرابط اللي درتلو كوبي من GitHub]
git push -u origin main

المرحلة 3: الربط مع Vercel (المرحلة الأخيرة)
-------------------------------------------
1. روح لموقع Vercel (vercel.com) ودير Login باستعمال حسابك نتاع GitHub.
2. اضغط على زر "Add New" ثم "Project".
3. راح تلقى الـ Repository اللي سميناه "solsniper" خرج لك، اضغط على "Import".
4. ما تبدل والو في الإعدادات، اضغط ديركت على زر "Deploy".
5. استنى دقيقة.. ومبروك عليك! السيت نتاعك راهو لايف (Live) وعندك رابط خاص بيك.

نصيحة ذهبية:
-----------
غير يولي السيت في Vercel، الـ CORS Errors راح يروحوا والـ Buy/Sell راح يولي يخدم 100% بلا مشاكل.
